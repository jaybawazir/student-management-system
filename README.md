# Student Management System

Text-based system in Python that manages students, courses, grades, schedules, and registrations through menus.

Built as a group project for Introduction to Programming, then split into separate files afterward so it's easier to navigate.

## Features

**Student Information**
- Display all students
- Add a new student
- Delete a student
- Edit a student's record

**Course Information**
- Display all courses
- Add a new course
- Delete a course
- Edit a course title

**Grades**
- View one student's grade in one course
- View all grades for one student
- View all students' grades in one course

**Schedule**
- Display a student's full course schedule
- Add a course and section to a student's schedule

**Registration**
- Display a student's registered courses
- Register a student in a course and section

## Files

- `main.py` — run this to start the program
- `data.py` — the sample data (students, courses, etc.)
- `students.py`, `courses.py`, `grades.py`, `schedule.py`, `registration.py` — one file per menu

## How to Run

```bash
python main.py
```

## Built With

Python 3 — lists, functions, loops, if/else
